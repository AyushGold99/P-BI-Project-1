Overview
This Power BI project presents a comprehensive Retail Analysis Dashboard that visualizes key performance indicators across multiple regions, product categories, and time periods. It enables decision-makers to monitor performance, identify trends, and drive strategic actions using data.

1) Key Features
Sales Performance: Track revenue, cost, and profit by region and product category.
Time Series Insights: Analyze monthly performance trends.
Region-wise Comparison: Drill down into regional KPIs.
Product Category Deep Dive: Understand product group-wise performance.
Interactive Filters: Use slicers to segment data by district, region, or time.

2) Tools & Technologies
Power BI Desktop
DAX (Data Analysis Expressions)
Data Modeling
Visualization Tools

3) File
Retail Analysis Project Sample.pbix: Power BI Desktop file containing the full dashboard and data model.

4) Visuals Included
Line charts for monthly trends
Bar and column charts for region and product analysis
Card visuals for KPIs (Sales, Profit, Units Sold)
Slicers for dynamic filtering
